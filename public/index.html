Principios de EmeraldJs

Estos son los 2 principios fundamentales de EmeraldJs.
EmeraldJs se enfoca en hacer el proceso de desarrollo agradable para el desarrollador sin sacrificar funcionalidad. Los desarrolladores mas felices escriben el mejor código. Para lograrlo, intentamos combinar lo mejor de lo que hemos visto en otro framework web como Ruby on Rails, ASP.NET MVC y Sinatra.

Creemos que desarrollar debe ser una experiencia agradable y creativa para ser plenamente satisfactorio, EmeraldJs es accesible, pero potente, proporcionando potentes herramientas para aplicaciones grandes y robustas.
